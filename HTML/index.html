HTML - Estruturando a Web

Introdução

É uma linguagem de marcação utilizada para dizer ao navegador como estruturar a página web que você visita. O HTML consiste em uma série de elementos que você usa para anexar, envolver ou marcar diferentes partes do conteúdo para que apareça ou aja de uma certa maneira.

O fechamento das tags pode transformar uma parte do conteúdo dentro do elemento em um link para direcionar à uma outra página web, colocar as paalvras em itálico, e assim por diante.

Anatomia De Um Elemento HTML

As partes principais do elemento são :
  1. Tag de Abertura : Consiste no nome do elemento, envolvido entre parenteses angulares de abertura e fechamento. Isso indica onde começa, ou inicia a produzir efeito.
  2. Conteúdo : Este é o conteúdo do elemento.
  3. Tag de Fechamento : É o mesmo que a tag de abertura , mas indica onde o elemento termina.
  4. Elemento : Tag de abertura + Conteúdo + Tag de Fechamento

Elementos em Block vs Elementos Inline

  • Elementos em Bloco = São como blocos de construção de uma página. Eles sempre começam em uma nova linha e ocupam todo o espaço disponível. Pense em parágrafos, listas ou títulos: cada um deles é um elemento em bloco. Esses elementos são usados para estruturar a página e organizar o conteúdo

  • Elementos Inline = São como as palavras em uma frase. Eles ocupam apenas o espaço necessário e não quebram a linha. Imagine um texto: as palavras, os números e os links são elementos inline. Eles se encaixam no fluxo normal do texto.

O que são Elementos Vazios ?

Elementos vazios são uma ferramenta útil para adicionar elementos simples e autocontidos a uma página HTML. Ao entender como eles funcionam, você pode criar código HTML mais conciso e eficiente.

Anatomia de um Documento HTML

Vamos aprender como estes elementos individuais são combinados entre si para criar uma página HTML inteira:

  1. <!Doctype HTML> = Identifica um documento HTML
  2. <html> = É a tag que envolve todo o conteúdo da página inteira (elemento raiz)
  3. <head> = É uma parte fundamental da estrutura HTML. Ele contém informações essenciais sobre a página que não são exibidas diretamente ao usuário, mas que são cruciais para o funcionamento e a visibilidade da página na web.
  4. <meta>
  5. <!title>
  6. <body>
